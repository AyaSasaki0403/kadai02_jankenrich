# ③課題番号-じゃんけんリッチアプリ

## ②課題内容（どんな作品か）

- JavaScriptを使って機械とユーザーがじゃんけんで勝負をし、勝敗によって役職が上下するゲーム。「社長」になると、メッセージが表示される。

## ③工夫した点・こだわった点

- PC画面にできる限り要素が納まるように、POPUPを活用しました。
- 1つのロジックだけでなく、複数のロジックが組み合わさって動作が生じるようにしました。
- 何度でもゲームを繰り返せるよう、POPUP windowを閉めると勝手にスコアが0になるように設定しました。

## ④難しかった点・次回トライしたいこと(又は機能)

- サイズが若干異なる画像が入れ替わると、レイアウトが崩れてしまう。時間がある際に、cssのレイアウトのことをもっと理解したいと思いました。
- 今回はPOPUPを複数活用することができたので、次回は別のアニメーションにトライしようと思います。
